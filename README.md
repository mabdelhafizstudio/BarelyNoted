# BarelyNoted

BarelyNoted is a sleek minimalist SwiftUI markdown notes app with a lavender Liquid Glass-inspired interface, sharing, multi-select deletion, and a burn-away delete animation.

The app is designed around Gentium Plus from Google Fonts. Add `GentiumPlus-Regular.ttf` and `GentiumPlus-Bold.ttf` to `BarelyNoted/Resources/Fonts/` when network access to Google Fonts or GitHub raw assets is available; the project already registers those font filenames in the generated Info.plist settings and uses `Font.custom("Gentium Plus", ...)` throughout.
